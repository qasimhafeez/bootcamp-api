# Bootcamp API
> Backend API for bootcamp and courses with complete user authentication.
## Usage

Rename "config/config.env.env" to "config.env" and update values and settings there.

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
